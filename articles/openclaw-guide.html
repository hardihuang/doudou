AI助手配置 · 2026-03-10

OpenClaw 完全指南

本文将详细介绍如何从零开始配置 OpenClaw AI助手，包括环境搭建、技能安装、多Agent协作等核心内容。

什么是 OpenClaw

OpenClaw 是一个开源的 AI Agent 框架，让你可以用聊天方式指挥 AI 完成各种任务。从发送消息、处理文档到控制浏览器，AI 助手可以成为你的数字员工。

环境搭建

1. 安装依赖

# 克隆仓库
git clone https://github.com/openclaw/openclaw.git
cd openclaw

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 文件，添加你的 API 密钥

2. 启动服务

# 开发模式
npm run dev

# 生产模式
npm start

配置 AI 助手

在 workspace 目录下创建以下文件：

IDENTITY.md - 身份信息

# IDENTITY.md

- **Name:** 豆豆
- **Creature:** AI assistant
- **Vibe:** 专业、可靠、高效，幽默风趣
- **Emoji:** ◉

SOUL.md - 行为准则

定义 AI 助手的行为方式、原则和边界。

安装技能

# 列出可用技能
openclaw skills list

# 安装技能
openclaw skills install feishu-doc
openclaw skills install web-search

多 Agent 协作

使用 sessions_spawn 功能可以创建多个 Agent 并行工作：

// 创建子 Agent 处理任务
const result = await sessions_spawn({
  task: "分析这份数据",
  agentId: "data-analyst"
});
