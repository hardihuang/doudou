# 课堂评价自动化系统 - 测试部署

> GitHub Pages 静态测试页面

## 🚀 部署说明

### 方式一：GitHub Pages（推荐）

1. 在 GitHub 创建新仓库，命名为 `keping-bot`
2. 上传 `index.html` 到仓库根目录
3. 进入 Settings → Pages → Source 选择 main 分支
4. 访问 `https://你的用户名.github.io/keping-bot`

### 方式二：直接访问

页面已部署，可直接访问测试。
