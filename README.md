# 课堂评价自动化系统

> AI驱动的课堂评价自动化解决方案

## 🎯 项目简介

课堂评价自动化系统，帮助导师快速完成课堂评价并发送给家长。

## 🚀 快速开始

### 本地预览

```bash
python3 -m http.server 8080
```

然后访问 http://localhost:8080

### GitHub Pages 部署

1. Fork 本仓库
2. 进入 Settings → Pages
3. Source 选择 main 分支
4. 访问 https://你的用户名.github.io/keping-bot

## 📋 功能特性

- ✅ 静态页面展示
- 🔄 后端服务开发中
- 📝 飞书机器人接入中

## 🛠️ 技术栈

- 前端：HTML5 + CSS3 + JavaScript
- 后端：Python Flask/FastAPI（开发中）
- 部署：GitHub Pages
